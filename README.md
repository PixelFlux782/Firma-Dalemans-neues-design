# Firmenwebsite – Stapelstuhl & Klapptisch

Dieses Projekt ist ein Next.js-Grundgerüst für die neue Homepage.

## Entwicklung starten

1. Abhängigkeiten installieren:

   ```bash
   npm install
   ```

2. Entwicklungsserver starten:

   ```bash
   npm run dev
   ```

3. Im Browser öffnen: http://localhost:3000
